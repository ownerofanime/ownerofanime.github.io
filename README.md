# Saver

A hyperlocal marketplace connecting nearby retailers with college students in Singapore. Retailers list near-expiry food items at discounted prices. Students discover, reserve, and pick up value deals near campus.

Built as an interactive prototype for the Retail Innovation Competition.

## What Each App Does

**Student App** — Deal discovery feed, interactive SVG map of nearby stores, deal detail with reservation and payment options, notifications, profile with impact stats.

**Vendor App (Saver Vendor)** — Dashboard with sales and waste metrics, deal posting and management, analytics with category breakdowns and actionable insights, store settings.

---

## Tech Stack

Self-contained single HTML files. No build step, no server.

- React 18 (loaded via CDN)
- Babel standalone (in-browser JSX compilation)
- Custom SVG icons throughout (no emoji, no external icon libraries)
- Hosted on GitHub Pages
